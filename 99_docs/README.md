# Analytics Project Package

This package updates `[Analytics]` so it keeps direct analytical capability while adding the universal main files standard.

## Files

- `PROJECT_INSTRUCTIONS.md` — main project instructions for `[Analytics]`.
- `knowledge/ANALYTICS_MAIN_FILES_STANDARD.md` — canonical `stage_main_full → mart_main_full → mart_main_tz/compact` standard.
- `knowledge/ANALYTICS_ROUTING_RULES.md` — routing between Analytics, Codex, Thinking, LLM, and AI OS.
- `knowledge/ANALYTICS_QA_ACCEPTANCE.md` — QA and acceptance checklist.
- `qa/SMOKE_QA_RESULT.md` — smoke QA result for this package.

## Installation

Copy files into the `[Analytics]` project knowledge/settings area.

Recommended placement:

```text
[Analytics]/
├── PROJECT_INSTRUCTIONS.md
└── knowledge/
    ├── ANALYTICS_MAIN_FILES_STANDARD.md
    ├── ANALYTICS_ROUTING_RULES.md
    └── ANALYTICS_QA_ACCEPTANCE.md
```
